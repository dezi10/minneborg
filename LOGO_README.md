## Bruksanvisning

### For Web
1. Logo er allerede integrert i alle HTML-sider
2. CSS styler logoen med hover-effekter
3. Responsive: 50px høyde (desktop), 40px høyde (mobil)

### For Visittkort
1. Åpne `business-card.svg` i Illustrator eller Inkscape
2. Eksporter til PDF eller PNG (300 DPI minimum)
3. Standard visittkort størrelse: 85x55mm
4. Trykk: Bruk profesjonell trykkeri med UV-coating for beste resultat

### For andre formål
- **Social Media**: Eksporter logo.svg til PNG 1000x1000px
- **Email signatur**: Eksporter til PNG 300x90px
- **Presentasjoner**: Bruk SVG direkte eller eksporter til høy-oppløsning PNG

## Tekniske detaljer

### SVG fordeler
- Skalerbar uten kvalitetstap
- Liten filstørrelse
- Editbar i kode
- Støtter gradienter og effekter

### Hover-effekter (Web)
- `transform: translateY(-2px)` - Løfter logo ved hover
- `filter: brightness(1.1)` - Øker lysstyrken
- Smooth transition (0.3s ease)
