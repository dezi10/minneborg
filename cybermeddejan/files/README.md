# Files - Dokumenter og Ressurser

Denne mappen inneholder nedlastbare filer og dokumenter tilknyttet porteføljen.

## Innhold

| Fil | Beskrivelse |
|-----|-------------|
| `OSI-modellen som verktøy for systematisk feilsøking i nettverk.pdf` | Fagartikkel om OSI-modellen (PDF-versjon) |

## Bruk

Filene i denne mappen kan lenkes til fra hovedsidene. Eksempel:

```html
<a href="files/OSI-modellen som verktøy for systematisk feilsøking i nettverk.pdf" download>Last ned PDF</a>
```

## Legge til nye filer

1. Plasser filen i denne mappen
2. Oppdater denne README med filbeskrivelse
3. Legg til lenke fra relevant side i `/cybermeddejan/`
